Exercise 6

In this exercise, I played around with the scenemanager script file where I just added a bunch of random stuff like text, images, and background colors for each of my 5 scenes. I also incorporated sounds, including one of my recorded voices "Hey," a blasting sound, and a relaxing medieval sound as background music for the first scene. The library I chose to use is p5.play(), as well as the primary libraries, p5.sound() and scenemanager.js. The main takeaway is to familiarize myself between the scene and sketch javascript files. The scene file adds the more detailed stuff into the scenes, and the sketch file sets the keyboard controls for changing the scenes. The p5.play library is what allowed the scene file to incorporate simple animations using png or jpg files.

Final Project

For my final project, I decided to create an interactive software that functions as a museum revolving around the basic historical facts of San Jose. It is a mashup between interactive software and museum software, and the software is intended to show the basic history and facts of San Jose that is tied to its connection with Vietnamese culture, as well as other landmarks and buildings. The timeline will consist of buttons that will each display their own unique historical fact about a particular place or landmark and maybe even some fun facts. I've used the p5.play() library to incorporate simple animations throughout the software.
