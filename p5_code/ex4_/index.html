Exercise 4

This exercise is a continuation of Exercise 3. This time, my rabbit characters are part of a class object that bounces back and forth on the canvas screen using the position and speed variables for x and y, known as ( posx, posy, speedx[sx], speedy[sy] ). More importantly, I've learned how to convert my rabbit character into a class object by introducing variables, the use of this in variables, and taking away the word function for functions inside class objects.

Controls for the Interactive:

  • B: Create a new angry rabbit character with yellow eyes. But beware that it moves fast when you do.
  • Left mouse: Hold the mouse to bring all of the rabbit arrays together and then release the mouse to let them roam around the canvas screen freely again from the dropped position.
