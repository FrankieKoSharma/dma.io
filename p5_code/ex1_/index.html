Exercise 1

In this very first coding exercise in Digital Media Art, I've experimented with the basics of JavaScript coding, such as background color, creation of basic shapes like rectangles and ellipses, console logs, if and else statements, variables, and the random function.

What I've created so far is that I've created a squared, colored grid that randomly changes colors in the background. Along with that, I've also set controls like when the mouse is on the right half of the canvas, it draws along using a blue ellipse, and on the bottom right quadrant, also draws an orange square, with both shapes emphasize heavily on the opacity feature.
